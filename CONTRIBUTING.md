# Contribuindo ao Projeto Madson Motors ERP

## Fases do Desenvolvimento

### Fase 1: Database & Setup
- Criar repositório GitHub
- Configurar Supabase PostgreSQL
- Executar scripts SQL
- Semear dados iniciais

### Fase 2: Backend
- Implementar API Node.js/Express
- Controllers, Routes, Services
- Autenticação JWT
- Integração com Supabase

### Fase 3: Frontend
- Interface React/Vite
- Componentes por departamento
- Dashboard e relatórios
- Chat em tempo real

## Regras Mandatórias
🔴 NUNCA colocar .env no GitHub
🔴 NUNCA pular validação de scripts
🔴 NUNCA deixar variável com placeholder
🔴 NUNCA executar script sem verificar o anterior funcionou

## Como Reportar Bugs
